Proyectos

FORMULARIO BASICO

Este proyecto es un formulario básico. Incluye campos de entrada para capturar información de usuario, como nombre, correo electrónico y mensaje. Está diseñado de forma sencilla, con un enfoque en la estructura y el diseño básico utilizando solo HTML y CSS.

MEMORAMA

Un juego de memoria interactivo donde los jugadores emparejan cartas iguales. Incluye animaciones visuales y un temporizador que registra el tiempo.

TO-DO LIST

Una aplicación para organizar tareas, con opciones para agregar, eliminar y marcar tareas como completadas. Permite filtrar tareas por estado (todas, completadas, pendientes) y guarda los datos localmente.

CLIMA ACTUAL

Una aplicación que muestra el clima actual de cualquier ubicación ingresada, utilizando datos en tiempo real de la API de OpenWeather. Incluye diseño responsivo, íconos dinámicos según las condiciones climáticas y una experiencia de usuario limpia.

PIEDRA, PAPEL O TIJERAS

Un juego clásico de Piedra, Papel o Tijera con un toque moderno: una IA que analiza las elecciones del jugador para mejorar su estrategia.
